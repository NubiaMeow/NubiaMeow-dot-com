umu-shortcut

I created this project to have a quick and easy way to create shortcuts for games installed in an UMU Proton prefix. It uses some (admittedly pretty janky) python code to create a desktop file in ~/.local/share/applications which points to a wrapper that sets the required environment variables.

Luchadora

Luchadora is a Unity game created for Global Game Jam 2026. We settled on a fun 2D-character-in-3D-environment style really early on in the jam which allowed the amazing artists I was working with to "do their thing".

This is a project I really hope to pick back up one day. It was an absolute blast working on it during the jam!
